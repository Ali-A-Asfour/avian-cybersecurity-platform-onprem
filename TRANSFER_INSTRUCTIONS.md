# Project Transfer Instructions

## Creating a Clean Source Zip

The project includes large build artifacts that should NOT be transferred:
- `.next/` - Next.js build cache (852MB)
- `node_modules/` - NPM dependencies (698MB)
- `.swc/` - SWC compiler cache
- `*.tsbuildinfo` - TypeScript build info

### Option 1: Use the Provided Script (Recommended)

```bash
./create-source-zip.sh
```

This creates a clean zip file (~15-20MB) with only source code.

### Option 2: Manual Zip Creation

```bash
zip -r avian-platform-source.zip . \
  -x "node_modules/*" \
  -x ".next/*" \
  -x ".swc/*" \
  -x "*.tsbuildinfo" \
  -x ".git/*" \
  -x ".env.local" \
  -x "storage/*" \
  -x ".workflow-state/*"
```

## After Extracting in New Repository

1. **Extract the zip file**
   ```bash
   unzip avian-platform-source-*.zip -d /path/to/new/repo
   cd /path/to/new/repo
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment**
   ```bash
   cp .env.example .env.local
   # Edit .env.local with your configuration
   ```

4. **Initialize database**
   ```bash
   npm run db:push
   ```

5. **Run development server**
   ```bash
   npm run dev
   ```

## What's Included in the Zip

✅ Source code (`src/`)
✅ Database schemas (`database/`)
✅ Configuration files
✅ Documentation (`docs/`)
✅ Scripts (`scripts/`)
✅ Docker configuration
✅ Package definitions (`package.json`, `package-lock.json`)

## What's Excluded from the Zip

❌ `node_modules/` - Install with `npm install`
❌ `.next/` - Generated by `npm run build`
❌ `.swc/` - Generated during build
❌ `.env.local` - Create from `.env.example`
❌ `storage/` - User uploads (empty in source)
❌ `.workflow-state/` - Session-specific state
❌ Build artifacts and caches

## File Size Comparison

- **With build artifacts**: ~1.5GB
- **Source only**: ~15-20MB (100x smaller!)

## Troubleshooting

### "Zip file is still huge"
Make sure you used the script or excluded the folders listed above.

### "Missing dependencies after extraction"
Run `npm install` in the extracted directory.

### "Build fails"
1. Delete `.next/` and `node_modules/`
2. Run `npm install`
3. Run `npm run build`

### "Database errors"
1. Check `.env.local` has correct `DATABASE_URL`
2. Run `npm run db:push` to sync schema
