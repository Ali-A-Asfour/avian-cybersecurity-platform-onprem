## Color Scheme

### Light Mode
- Background: White (#FFFFFF)
- Cards: White with subtle shadow
- Text: Dark gray (#111827)
- Secondary text: Medium gray (#6B7280)
- Primary buttons: Blue (#3B82F6)
- Success: Green (#10B981)
- Error: Red (#EF4444)
- Borders: Light gray (#E5E7EB)

### Dark Mode
- Background: Dark gray (#1F2937)
- Cards: Darker gray (#111827)
- Text: White (#FFFFFF)
- Secondary text: Light gray (#9CA3AF)
- Primary buttons: Blue (#3B82F6)
- Success: Green (#10B981)
- Error: Red (#EF4444)
- Borders: Dark gray (#374151)

## Interactive Elements

### Toggle Switches
- Off state: Gray background, white circle on left
- On state: Blue background, white circle on right
- Smooth animation on toggle

### Buttons
- Primary: Blue background, white text
- Outline: White/transparent background, colored border
- Hover: Slightly darker shade
- Disabled: Reduced opacity, no hover effect

### Form Inputs
- Border: Light gray (light mode) / Dark gray (dark mode)
- Focus: Blue ring around input
- Error: Red border and error message below

### Status Badges
- Connected: Green background
- Disconnected: Gray background
- Error: Red background

## Responsive Behavior

### Desktop (>768px)
- Full width layout with max-width constraint
- Side-by-side timezone and language selectors
- Three-column grid for notification types

### Mobile (<768px)
- Stacked layout
- Full-width form inputs
- Stacked timezone and language selectors
- Single-column notification types
- Hamburger menu for tabs

## Accessibility Features

- Keyboard navigation support
- ARIA labels on interactive elements
- Focus indicators on all interactive elements
- Screen reader friendly
- High contrast mode support
- Semantic HTML structure
